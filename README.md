## Description
SimplyDone is a streamlined, aesthetic task management application designed to bring clarity to your daily workflow. 

## Technology Stack
*   **Frontend**: React 19, TypeScript, Tailwind CSS
*   **Architecture**: Service-Controller Pattern (Simulated Backend)
*   **Persistence**: LocalStorage (JSON Database)

## Project Stats
*   **Approximate Lines of Code**: ~900
*   **Number of Files**: 16
*   **Source Attribution**: New project created for dataset

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
